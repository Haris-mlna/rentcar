## PROJECT
  WebApp Cars Rental
  1. Auth Registration :
    - User : {
      nama : string,
      alamat : string,
      phone : number,
      SIM : number
    }
    - Dashboard
  2. Car Management :
    -Add Car
    -List Car (dynamic pages for each car)
    -Search Car
  3. Rent Car Management :
    -Rent Car
    -Verification / Validation
    -List Car Rented
  4. Return Car :
    - Return Car
    - Verification & Validation
    - Price
  5. Logout

Frontend (Angular 16.10):
  - Angular 16.1.0
  - Node.js 20.2.0
  - NPM 9.6.6
  - jwt

Backend (Laravel 10.10) :
  - Laravel framework 10.10
  - Laravel sanctum 3.2
  - Laravel tinker 2.8
  - php 8.1
  - Xampp with php (8.2.8)
